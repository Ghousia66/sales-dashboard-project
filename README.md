## Power BI Dashboard

### Overview
Interactive dashboard built using Power BI to visualize sales performance.

### Data Cleaning & Transformation
- Removed invalid date records causing parsing errors
- Converted Order Date into proper date format using Power Query
- Handled missing/error values before visualization

### Features
- KPI: Total Sales
- Sales by Region
- Top 5 Products
- Monthly Sales Trend
- Interactive filters (Region, Category)
